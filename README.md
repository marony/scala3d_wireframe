# 仕様

## 座標系

- 右手座標系が主流なんて知らなかったので、左手座標系
  - x軸は左から右
  - y軸は下から上
  - z軸は手前から奥
- ポリゴンは半時計回りが表(怪しい)

## 処理の流れ

- オブジェクトの回転
- 拡散光の計算
- ビューポート変換
- カリング
- 奥からソート
- 射影変換
- スクリーン変換
- 描画
